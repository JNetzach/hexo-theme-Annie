专注力(concentration)专注力是使用注意力的能力。 专注力比智商重要。 注意力是学习的必要条件。 注意力是对信息的选择。 普通信息到达感官(听、见、触觉)就停止。 只有充分注意的信息，会从感官形成概念，再从概念形成解读，一直到前额叶皮质 前额叶皮质有大量的神经元能够长时间的激活，信息进入这里才能记住和学习。 大脑三种注意力：注意什么，什么时候注意、如何注意 刺激越激烈学得越快 如果你对你学习的东西有强烈的热情和兴趣，那么这个时候，就会产生强烈的兴奋情绪，大脑就会更加活跃，可塑性更好，学习得更好。 专注力是选择注意什么就能看到什么，没看到的就会被忽略。 如何注意，靠的是执行控制系统 执行控制系统跟智商有关 执行控制系统的强弱体现在工作记忆能记住多少东西，能同时用多少个东西思考，执行速度快不快，能不能高效完成一个高强度逻辑项目 每接受一年正规教育，能增长1-5的智商 专注控制，做该做的事，不做不该做的事 提高专注力：耐住性子控制身体、思想，执行固定流程。 Become a powerful Software Engineer

大脑一切的知识和技能都是于神经元连接的形式存在于大脑的。 神经元身上细长又较粗、作为传输电信号的主干道是轴突，包裹着轴突的是髓鞘，轴突的外端点用于连接神经元的是突触。 大脑中有几百亿神经元，彼此间连接，构成了无比复杂的网络。 知识、能力、记忆都由其中的子网络实现。 知识就是神经元的互相连接 学习就是强化连接，不是创造新的连接 学习就是神经元之间连接，神经元越活跃，代表连接的强度在加强，信号传递的速度越快越精准 记忆有四种 工作记忆：也叫短期记忆，很快就会忘记，类似计算机的内存 情景记忆：每日的生活，越特别的，记得越好，兴奋、多巴胺分泌，会加强神经元的连接，但对不是特别的情景，过后不回忆，也会忘得很快 语义记忆：也就是长期记忆，睡觉时，海马体会把白天的情景记忆输送进大脑皮质中，重新编码，变成长期记忆，所以睡眠对记忆是很重要的 进程记忆：也叫肌肉记忆、内隐记忆，它记住的不是知识点，是一段动作，如弹钢琴、打字等，不经过海马体，直接在基底神经节存储 学习，就是想要语义记忆，进程记忆，加强神经元连接 Become a powerful Software Engineer

自学(Learning)前面说了自学需要三个基本，自信心和自我管理能力是两个基本。 现在说最后一个，文本能力。 文本能力，分三级。 第一级：抓重点，阅读一段文字，一篇文章，上一节课，要能抓到重点。 这是基本功。你看什么重点，首先得看你脑子里有什么，同一段文字，不同的人看，得到的是不同的东西。 你需要抓到重点以后，和你已有的联系起来，才能算有基本功。 练习这个基本功，方法是 “记笔记”。 右边写重点，左边写线索，下边写总结。 练习做到，看见总结想到线索，看到线索能想到重点。 第二级：有问题，能从一堆书中找到想要的答案。 有问题，不要局限于几本书上找答案，书只是指向月亮的手指，可以多参考几根手指，找到更好的方向。 第三级：建立自己的系统 一开始就规划好大纲，学习这门课的规划，要从中得到什么 把知识记下来，或者制定成一本书，让自己学的知识有体系。 用自己的语言，根据自己的思路规划，整理章节、专题，心得、使用技巧。 要简洁、清晰的直指重点，不要花里胡哨。 自我检验学习是为了应用。 学到一定程度，可以自己测试，或者参加比赛，检验自己的所学。 最后，让自己对这门学问Mastery ...

自学者(autodidact)以“自我”为主的态度“自学”。 当下，获取信息已经足够方便，剩下的只需本人自立，付诸行动，自学，就是最根本的学习方法。 The Success Pyramid： confidence(自信心) self-management(自我管理) learning 首先，自信：人在面对威胁的时候会让身体和头脑进入封闭状态，在次状态下人是无法学习新知识的，所以首先自己必须对学习的东西要有足够的信心。 其次，自我管理能力：对学习的安排、组织、计划、时间管理、行动能力……。准备学什么，就需要自己调查用什么书，什么材料等等。 最好，用一个文件夹，把所有资料、练习内容安排好，制定一个学习计划，安排时间，最后把学习成果制作成一份报告。 养成良好的学习习惯，不要拖延。 考核自己的水平。 Become a powerful Software Engineer PS : done.
